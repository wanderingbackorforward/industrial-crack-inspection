# Data

## Expected Directory Structure

```
data/
├── raw/               # Original images, masks, and annotations
├── processed/         # Train/val/test splits generated by scripts/prepare_dataset.py
│   ├── train/
│   │   ├── images/
│   │   ├── masks/
│   │   └── annotations/
│   ├── val/
│   └── test/
```

## Detection Format (COCO-like JSON)

Each `annotations/{basename}.json`:

```json
{
  "annotations": [
    {
      "bbox": [x, y, width, height],
      "category_id": 1,
      "width_grade": 0.2
    }
  ]
}
```

## Segmentation Format

- `images/{basename}.jpg`: RGB image.
- `masks/{basename}.png`: single-channel mask where crack pixels are > 0.

## Public Datasets

Compatible public crack datasets:

- CrackSeg9k
- CrackTree206
- SDI-CRACK
- CrackSegNet
- Bridge Crack Library

Convert them to the above format and run `scripts/prepare_dataset.py`.
